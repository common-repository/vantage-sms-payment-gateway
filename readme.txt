== Description ==
This plugins adds a new payment gateway for Vantage. This is only for Romanian networks


== Installation ==

1. Upload `plugin-name.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Go to http://grinmedia.ro/clienti/cart.php?a=add&bid=4 and order the Bundle (or you can just order soft si module WHMCS - Reseller plati SMS and Gratuitati - Vantage SMS Payment Gateway. Both are free and required)
4. Fill the settings form in the back-end of your Vantage
5. Start earning money!!

== Frequently asked questions ==

This plugin works only with romanian mobile networks. So, is very useful in Romania :)).  Before using the plugin you need to go to http://grinmedia.ro website, register and request a Reseller SMS service. The service is free, and you will be provide you with an unique ID to track your sales. 
- We also provide SMS service for Paygol.com and Mobio.bg
